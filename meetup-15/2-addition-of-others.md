# Addition of Others

## Summary

Write a function that takes an array of integers and returns an array of the sums. The sums must be the sum of all the numbers in the array EXCEPT for the current value.

## Input

Array/list of numbers/integers.

## Output

Array/list of numbers/integers.

## Example

Input: `[1, 2, 3, 4]`
Outupt: `[9, 8, 7, 6]`
